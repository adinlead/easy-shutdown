# 易关机

#### 介绍
一个超简易的关机程序

#### 软件架构
本工具有两个实现方式：
javafx包中的使用JavaFX实现
swing包中的使用swing实现


#### 使用说明

1.  使用`mvn clean package`编译代码得到Jar包`target\EasyShutdown-1.0-SNAPSHOT.jar`
2.  使用`java -jar target\EasyShutdown-1.0-SNAPSHOT.jar`运行程序
